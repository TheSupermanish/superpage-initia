# InitPage Demo Script (1-3 min)

## Opening (10s)

"InitPage is an AI-native commerce platform running on its own Initia appchain. Agents and humans buy, sell, and monetize digital resources with instant USDC payments."

---

## 1. Show the Appchain (20s)

- Open Initia Scan → show "initpage" rollup
- Show blocks being produced, contract deployments
- "This is our own Initia MiniEVM rollup — every transaction generates revenue for the platform."

## 2. Connect Wallet (20s)

- Click "Connect Wallet" on init.superpa.ge
- Use **social login** (Google/email) — no MetaMask needed
- "InterwovenKit gives users social login — no wallet setup required."
- Show the .init username display in the navbar

## 3. Enable Auto-Sign (15s)

- Open wallet dropdown → click "Auto-sign: OFF" to toggle ON
- "Session keys let agents transact without wallet popups. Sign once, trade freely."

## 4. Browse & Purchase (30s)

- Go to Explore page
- Show the "Bridge from any chain" banner
- Click a resource (Weather API - $0.50)
- Purchase modal opens → shows "⚡ Instant purchase" badge
- Click "Pay 0.50 USDC"
- Payment confirms instantly (no wallet popup because auto-sign)
- Content delivered — show the live weather data

## 5. Show On-Chain Proof (15s)

- Show the transaction hash
- Open Initia Scan → find the TX
- "Every purchase is verified on-chain on our appchain."

## 6. Agent Surface (20s)

- Show Claude Desktop with MCP tools (12 tools)
- Or show Telegram bot
- "Agents can discover, purchase, and access resources autonomously through MCP, A2A, or Telegram."

## 7. ERC-8004 Identity (10s)

- "Agents build on-chain reputation through ERC-8004 — identity, feedback, and validation registries deployed on our chain."

---

## Closing (10s)

"InitPage — agent commerce on its own Initia appchain. Session keys for frictionless transactions, Interwoven Bridge for cross-chain deposits, and .init usernames for identity. Built for the era of AI commerce."

## Key Talking Points

- **Own appchain** = every transaction is platform revenue, no gas leakage
- **Session keys** = agents transact autonomously, no wallet popups per purchase
- **Interwoven Bridge** = users deposit from any Initia ecosystem chain
- **.init usernames** = human-readable agent identities
- **x402 protocol** = HTTP 402 payment-gated resources (industry standard)
- **ERC-8004** = on-chain agent identity and reputation
- **12 MCP tools** = Claude Desktop can buy resources autonomously
